# Projetos

Como o CYD é uma plataforma comum, ele se torna muito útil para compartilhar projetos. Esta página será uma lista de projetos disponíveis para o CYD.

## Aviso!

A presença de projetos nesta página não representa necessariamente uma aprovação minha. Não testarei cada projeto adicionado, portanto instale esses projetos por sua própria conta e risco!

## Adicionando um projeto

Se tiver um projeto que gostaria de adicionar, fique à vontade para incluí-lo na lista!

Novos projetos devem ser adicionados ao final da lista.

Algumas regras:

- O projeto deve ser open source
- O projeto deve ser funcional - não há problema se ainda não estiver concluído, mas ele deve fazer o que afirma fazer!
