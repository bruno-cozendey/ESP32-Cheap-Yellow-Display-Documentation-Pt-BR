# Adaptador SD para NFC
Esta é uma PCB para disponibilizar os pinos do cartão MicroSD. Ela foi projetada especificamente para o módulo NFC PN532 vermelho, mas pode ser usada para qualquer finalidade.

![IMG_20231130_123558](https://github.com/witnessmenow/ESP32-Cheap-Yellow-Display/assets/1562562/3c1b8149-c26c-49c4-be1b-03e8a84d07db)


## Qual espessura de PCB pedir

Cartões SD geralmente têm 0,75 mm de espessura, mas essa não é uma espessura típica de PCB oferecida pelas fabricantes.

O SD card sniffer que comprei no Aliexpress tinha 1 mm de espessura.

Encomendei PCBs de 0,8 mm e de 1 mm da JLC, e ambas funcionaram bem.
